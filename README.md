# InSARTools

A Python toolkit for processing InSAR-derived deformation products.

## Supported Sensors

- Sentinel-1A
- Sentinel-1B
- Sentinel-1C
- Sentinel-1D
- ALOS PALSAR
- ALOS-2 PALSAR-2

## Supported Processing Software

- ISCE2
- MintPy
- ROI_PAC

## Current Modules

- Phase → LOS conversion
- Geocoding
- Coherence masking
- Reference correction

## Future Modules

- LOS decomposition
- GNSS integration
- Quadtree downsampling
- Slip inversion
